# Playa

Playa provides a lightweight library and CLI for playing audio using the host's
installed command-line players. The library focuses on detection and player
selection, while the CLI provides a simple `playa` command and metadata display.

## Components

- Library: `playa/lib`
- CLI: `playa/cli`

- Library details: `playa/lib/README.md`
- CLI details: `playa/cli/README.md`

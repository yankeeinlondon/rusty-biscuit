Rusty Biscuit Monorepo
======================

| Module | Description |
| ------ | ----------- |
| biscuit-hash | Provides best-in-class hashing (xxHash, blake3, argon2) |
| biscuit-speaks | Provides TTS functionality by leveraging the underlying host as well as cloud APIs |
| biscuit-terminal | Metadata extraction from the terminal as well as components to help writing to terminal |
| darkmatter | A library and CLI to render and transform markdown |
| playa | Library and CLI to play audio, leveraging host's resources |
| queue | Small TUI for queuing tasks; used primarily to avoid AI usage caps |
| research | Library and CLI for performing research and deploying to Agentic CLIs |
| schematic | A set of packages used to generate type safe API clients |
| sniff | A library and CLI for detecting hardware, network, and software resources on a host |
| so-you-say | Simple TTS client leveraging `biscuit-speaks` |
| tabby | A way to package a multi-page app into a single file for documentation and distribution |
| tree-hugger | Cross-language static analysis tool |

# EMQX Broker API


The [EMQX Broker](https://github.com/emqx/emqx) provides a full REST API. Use the `emqx` skill to implement this API in the `schematic` package.

- use primitives in `schematic/define`
- define the API in `schematic/definitions`
- use the `schematic/gen` package to generate end-user schema in `schematic/schema`

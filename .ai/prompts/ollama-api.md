# Ollama API

Use the `ollama` skill and then refresh you're memory of the **schematic** package in this monorepo.

- you will use the primitives in `schematic/define` to define the Ollama API in the `schematic/definitions` package
- you will then use the `schematic/gen` package to generate a schema in `schematic/schema`
- always use the `rust` skill, always use `rust-testing` for any tests

mat CLI

A CLI markdown renderer for the terminal.

Syntax

mat <file> [options]

The mat command will render a markdown file with rich color themes.

Options

  • --html - instead of showing it in the terminal it will be rendered to HTML (in a temp directory) and opened in your default browser

  • --theme - by default it will use the one-dark theme if your terminal is in dark mode (as it should be) or one-light theme if you're preparing on moving to the sun. Other options are:

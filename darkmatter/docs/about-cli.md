# About Darkmatter CLI

The **Darkmatter** CLI is a **Markdown** renderer which renders:

- [CommonMark](https://commonmark.org) Markdown
- will full support for [Github For Markdown(GFM)](https://github.github.com/gfm/) extensions
- but it also supports a small DSL we call [**Darkmatter**](./darkmatter-dsl.md)

# Research Filesystem

- to be a known research topic, the research must be located in `${research}/${type}/` directories. Where:
    - `research` directory is by default `${HOME}/.research` but can be set explicitly with the `RESEARCH_DIR` environment variable.
    - and the `type` subdirectory is one of the _types_ of research the CLI can produce (currently the only type is "library")

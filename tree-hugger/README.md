# Tree Hugger

A Library (and CLI) for generating diagnostics on a number of programming languages using static analysis (using the `tree-sitter` library).

## Supported Languages

- Rust 
- Javascript 
- Typescript 
- Go 
- Python 
- Java 
- PHP 
- Perl 
- Bash 
- Zsh 
- C 
- C++ 
- C# 
- Swift 
- Kotlin 
- Scala (`tree-sitter-scala`)
- Lua (`tree-sitter-lua`)
